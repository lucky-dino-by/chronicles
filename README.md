# Chronicles

A minimalist digital logbook application built with React, TypeScript.

## Features

Stores entries in localStorage (no network required)

- **Mobile-First Design**: Optimized for iPhone/Pixel devices
- **Minimalist Aesthetic**: High contrast black & white theme with monospaced typography

## Setup

1. Install dependencies:
```bash
yarn install
```

2. Run the development server:
```bash
yarn dev
```

3. Build for production:
```bash
yarn build
```

## Project Structure

Project is organizedusing a 'Folder by Feature'.

## Usage

1. **Local Mode** (default): All entries are stored locally in your browser's localStorage
